# hfut-reservation-admin

### 面向校园公共资源预约与使用服务的微信小程序的设计与实现

#### B 端后台 @vue/cli

#### 项目目录结构说明

- `dist` 打包构建目录
  - `dev` 开发模式打包目录
  - `build` 生成模式打包目录
- `node_modules` 项目依赖的第三方模块
- `public` 网页应用公共文件目录
- `src` 源文件目录
  - `config` 配置文件目录
    - `index.js` 默认配置文件（其中包含开发环境配置）
  - `views` 页面文件
  - `store` 状态管理
  - `styles` 静态公共样式
  - `utils` 存放自己封装或第三方的小工具
  - `main.js` 项目入口文件
- `package.json` 项目配置文件
